# terraform-aws

Ansible And Terraform scripts for the Gophie Infrastructure

## Requirements
- Terraform
- [Terraform Namecheap](https://github.com/adamdecaf/terraform-provider-namecheap)
- AWS CLI (or Credentials)
- Ansible

## Create Infra
```bash
  $ terraform apply
```


## Destroy Infra
```bash
  $ terraform destroy
```
